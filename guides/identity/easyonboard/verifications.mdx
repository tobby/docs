The Verifications results page is where businesses monitor all onboarding activities. It is designed to store and display all verifications conducted via the Identity Verification Widget. It provides a comprehensive view of each verification process, allowing businesses to easily access and review verification details.

The Verification Detailed Page includes the following sections:

1. **Verification Activity Information:** This includes all the activities tracked for each verification, and they include:

	- **Reference ID:** The unique identifier for the verification process.

	- **Total Time Spent:** The total duration of the verification process.

	- **Number of Sessions:** The total number of sessions involved in the verification.

	- **Date/Time Created:** The date and time when the verification process was initiated.

	- **Total Cost:** The total cost associated with the verification process.

	- **Cost Breakdown:** A detailed breakdown of the costs incurred during the verification.

	- **Activity Log:** A log of all activities and interactions during the verification process.

2. **Bio Data:** This section contains all the user verification details collected via the widget, such as government data, uploaded government-issued ID, user data, and other verification pages.  


3. **IP/Device Check:** This section displays detailed information about the device used during the verification process.


4. **Address Verification:** This section shows the details of the user's address verification.


5. **AML Check:** This section provides information on the user's Anti-Money Laundering (AML) check results.
