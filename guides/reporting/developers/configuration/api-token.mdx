API Tokens allow you to authenticate and authorize API requests securely within your application.


1. Enter a token name to identify the token

2. Select the App it should be associated with

3. Click on the Create Token button


#### For security:


- Tokens are hidden by default and can only be revealed after entering your account password.

- Do not expose your token in public repositories.

- If you suspect your token is compromised, delete it immediately and create a new one.
