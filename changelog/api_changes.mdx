The API changelog lists the changes to the Dojah API, SDKs and libraries. If you’re looking for information about feature additions and product updates, see the [product blog](https://dojah.io/blog).

The changelog below reflects new product developments and updates on a monthly basis.

## July 2025

**1. Email Notifications Rolled Out**

You can now add multiple team members to receive important email alerts. This ensures everyone stays informed and can act swiftly on:

- Balance Depletion Alerts
- System Updates & Critical Notifications


**How it works:**

- Navigate to the Settings > Email Notifications tab
- Select relevant team members and add them to the email notification list
- Team members can now easily receive important notifications

![](/images/july_2025_1.png)


**2. Revamped API Documentation**

We’ve restructured and improved the API documentation for a more intuitive and developer-friendly experience:

- Clearer Endpoint Descriptions
- Improved Navigation & Categorization
- Detailed Integration Examples
- This update helps developers discover, understand, and integrate faster, with less back-and-forth.
