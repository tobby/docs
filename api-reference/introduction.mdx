Dojah offers you a wide range of services that allows you to easily build ground breaking product and services  
As a one stop shop for APIs , we give you access to connect to several APIs for different services with one integration.

We will provide you with detailed information on how to easily consume our APIs and use your keys to make API calls directly from our documentation.

### Getting started

**To get started with our APIs;**

1.  You need to create an account ([https://app.dojah.io/signup](https://app.dojah.io/signup)) on Dojah.

### API Keys Access

To access the API Keys;

1.  You must have created an account
2.  You will be provided with the public and private API keys for the sandbox and production environment respectively.
3.  The private key is for connecting to all Dojah API endpoints
4.  3.  The public key is required to use the KYC widget
