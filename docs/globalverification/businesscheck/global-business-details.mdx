---
title: "Global Business Details"
description: "Get details of a particular business"
---



### Request
```http [GET]
{{baseUrl}}/api/v1/kyb/business/detail
```

### Header

| Header | Type | Description |
| --- | --- | --- |
| AppId | string | Create an app to get your app ID on dashboard [here](https://app.dojah.io/developers/configuration)  |
| Authorization | string | private key |

### Query parameter

| Parameter | Type | Description | Required |
| --- | --- | --- | --- |
| international\_number \* | string | International number of company | required |
| country\_code \* | string | Country Code of business country | required |
