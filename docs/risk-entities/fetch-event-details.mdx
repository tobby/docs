description: ""
---

The Fetch Event Details endpoint lets you retrieve the full payload and decision data for a previously ingested event.

### Endpoint

    GET /api/events/{event_id}
    Authorization: {PRIVATE_KEY}

-   `event_id` is the identifier assigned or echoed by ProfiledRisk for the evaluated event
    

### Parameters

Name

Type

Description

`event_id`

string

Unique identifier of the event you want to retrieve
