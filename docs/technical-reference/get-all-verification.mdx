---
title: "All Verification"
icon: "book"
description: "Get the full list of all the verifications on applications"
---


### Request

```http [GET]
{{baseUrl}}/api/v1/kyc/verifications
```


### Header

| Header | Type | Description |
| --- | --- | --- |
| AppId | string | Create an app to get your app ID on dashboard [here](https://app.dojah.io/developers/configuration)  |
| Authorization | string | public secret key  |


### Query parameter

| Parameter | Type | Description | Required | 
| --- | --- | --- | --- |
| term  | string | for search with email, reference_id | optional |
| start  | string | Start date filter e.g YYYY-MM-DD| optional |
| end  | string | End date filter e.g YYYY-MM-DD | optional |
| status  | string | status filter e.g Ongoing, Completed, Pending, Failed | optional |
