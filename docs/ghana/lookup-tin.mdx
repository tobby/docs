---
title: "Lookup Ghana TIN"
icon: "file-code"
description: "This endpoint allows you to Lookup a Ghana Tax Identification Number for individuals. For businesses see Ghana Business Lookup under Dojah Business."
---


### Request

```http [GET]
{{baseUrl}}/api/v1/gh/kyb/tin?tin=C00000000021
```

### Header

| Header | Type | Description |
| --- | --- | --- |
| AppId | string | Create an app to get your app ID on dashboard [here](https://app.dojah.io/developers/configuration)  |
| Authorization | string | public secret key  |

### Query parameter

| Parameter | Type | Description | Required |
| --- | --- | --- | --- |
| tin \* | string | The TIN number | required |

### Sample response

```json Response
{
  "entity": {
    "is_valid": true,
    "regn_status": "REGISTERED",
    "rgdno": "12345678",
    "tax_office": "ACCRA",
    "taxpayer_name": "JOHN DOE",
    "tin": "C1234556787",
    "trading_as": "123R43A"
  }
}
```
