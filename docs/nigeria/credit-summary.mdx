---
title: "Credit Summary"
description: "The Credit Summary API provides a comprehensive summary of an individual's credit history, linked to their Bank Verification Number (BVN). This API is designed to give financial institutions, lenders, and credit agencies a quick and accurate overview of a user's creditworthiness, including outstanding loans, repayment history, and overall credit behavior."
---


Use with the BVN Number

### Request

```http [GET]
{{baseUrl}}/api/v1/credit_bureau
```

### Header

| Header | Type | Description |
| --- | --- | --- |
| AppId | string | Create an app to get your app ID on dashboard [here](https://app.dojah.io/developers/configuration)  |
| Authorization | string | public secret key |

### Query parameter

| Parameter | Type | Description | Required |
| --- | --- | --- | --- |
| bvn | string | BVN Number of Customer | required |
