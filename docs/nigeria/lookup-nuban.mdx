---
title: "Lookup NUBAN"
description: "The Lookup NUBAN (Nigeria Uniform Bank Account Number) endpoint provides information of users' account numbers"
---

### Request

```http [GET]
{{baseUrl}}/api/v1/kyc/nuban
```

### Header

| Header | Type | Description |
| --- | --- | --- |
| AppId | string | Create an app to get your app ID on dashboard [here](https://app.dojah.io/developers/configuration)  |
| Authorization | string | public secret key |

### Query parameters

| Parameter | Type | Description | Required |
| --- | --- | --- | --- |
| account_number \* | string | Bank account number | required |
| bank_code \* | integer | Bank Code | required |
