### Lookup NIN Advance 

### Request

```http [GET]
{{baseURL}}/api/v1/kyc/nin/advance
```

### Query parameters

| Parameter | Type | Description | Required |
| --- | --- | --- | --- |
| nin \* | string | A valid nin | required |
