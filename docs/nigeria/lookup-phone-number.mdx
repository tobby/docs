## Phone Number Basic

```http [GET]
{{baseUrl}}api/v1/kyc/phone_number/basic

```

### Header

| Header | Type | Description |
| --- | --- | --- |
| AppId | string | Create an app to get your app ID on dashboard [here](https://app.dojah.io/developers/configuration)  |
| Authorization | string | public secret key |

### Query parameters

| Parameter | Type | Description | Required |
| --- | --- | --- | --- |
| phone_number \* | string | A valid phone number | required |

### Sample response


```json Response
{
    "entity": {
        "first_name": "JOHN",
        "middle_name": "DOE",
        "last_name": "CHHUKWU",
        "gender": "Male",
        "nationality": "NGA",
        "date_of_birth": "1990-05-16",
        "msisdn": "23481222222222"
    }
}
```
