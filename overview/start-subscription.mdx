After signing up, you will need to select a subscription plan to activate the platform's features.


<Card title="Steps to Start a Subscription" icon="link">
 

1.  Navigate to the Billing section, and then select the “Add New Card“ button.
    
2.  Review the available subscription tiers (Basic, Premium, Enterprise).
    
3.  Compare the plans using the detailed comparison table, which outlines features across:
    
    -   **Core Usage:** (e.g., Monthly event limits)
        
    -   **Feature Functionality:** (e.g., Access to specific dashboard tools)
        
    -   **Integration:** (e.g., API access levels)
        
    -   **Support:** (e.g., Level of customer support)
        
4.  Select the plan that best fits your business needs and follow the prompts to enter your payment details and confirm your subscription.



  </Card>
